-------------------------------------------------------------------------------------
 DESCRIPTION
-------------------------------------------------------------------------------------
        Airwick Menu est un script à inclure dans un script PHP, celui-ci permet de
        générer un menu type application windows, horizontal et déroulant. Ce script
	utilise le PHP, l'HTML, le JavaScript et le DHTML (calques, CSS...)

-------------------------------------------------------------------------------------
 POINTS FORTS
-------------------------------------------------------------------------------------
	+ Reproduit le menu Windows : ergonomie garantie !
	+ Compatible avec la plupart des navigateurs, saufs options spéciales
	+ Stockage des menu dans une base de données
	+ Administration simple

-------------------------------------------------------------------------------------
 CONFIGURATION REQUISE
-------------------------------------------------------------------------------------
	- Serveur d'application supportant le PHP (testé sur 4.05)
	- Serveur MySql (Testé sur MySql 3.23.32)

-------------------------------------------------------------------------------------
 MISE A JOUR
-------------------------------------------------------------------------------------

	1 - Executer éventuellement les commandes SQL du fichier
	    maj-x.x.x-x.x.x-sql.txt sous une console MySql ou PHPMyAdmin.
	2 - Remplacer ou ajoutez les fichiers suivants :
		
		- config_menu.php (Attention à vos anciens paramètres !!!)
		- admin_menu.php
		- airwick_menu.php
		- barre01.gif
		- barre02.gif
		- barre04.gif
		- plus01.gif
		- plus05.gif
		- plus06.gif
		

	3 - Executez le script admin_menu.php et effectuez eventuellement des
	    modifications.
	4 - CLIQUEZ SUR LE BOUTON GENERER DU SCRIPT D'ADMINISTRATION !

-------------------------------------------------------------------------------------
 INSTALLATION
-------------------------------------------------------------------------------------

	1 - Créez les tables selon le shéma du fichier tables-sql.txt.
	2 - Il y a déjà des données d'exemple, vous pouvez les enlever
	    si vous le désirez.
	3 - Changer l'accès à la base de données dans le fichier config_menu.php
	4 - Changez les couleurs, la police... grâce au fichier config_menu.php
	    Si vous changez les paramètres de position du menu, veuillez éxecuter
	    la admin_menu.php et cliquer sur le bouton 'Génerer'
	5 - Créez vos menu grâce au fichier admin_menu.php, n'oubliez pas d'en 
	    restreindre l'accés !!!
	6 - Insérez ce code grâce à la fonction PHP "include". Attention ! Ce
	    script doit s'insèrer dans le corps de la page HTML : entre les
	    balises <BODY> et </BODY>
